Il faut exécuter la classe DIROgueServer, puis la classe DIROgueClient afin de se connecter au serveur. 

Ensuite, entrer la commande "load", puis indiquer le chemin absolu vers le fichier de commandes command.txt. 

Exécuter la classe App, puis cliquer sur le bouton "Load". Choisir ensuite le fichier rapport.txt. 

Lorsque le rapport s'affiche à l'écran, cliquer sur le bouton "Replay". Ceci affichera la vue de l'aventure.

Les boutons Forward et Backward servent à naviguer dans l'aventure, et le bouton Exit pour quitter.
